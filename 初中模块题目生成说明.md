# 初中模块自选20题模式题目生成说明

## 概述

已为35个三级知识点各生成2个选择题和1个填空题，共105道题目。

## 文件说明

- `generate_middle_school_questions.js` - 题目生成脚本（包含完整的题目模板）
- `middle_school_questions.json` - 生成的题目JSON文件（运行脚本后生成）

## 使用方法

### 方法1：运行Node.js脚本（推荐）

```bash
# 在项目根目录下运行
node generate_middle_school_questions.js
```

脚本会自动：
1. 读取语法数据结构
2. 为每个三级知识点生成2个选择题和1个填空题
3. 添加考频信息（基于2023-2025年真题）
4. 保存到 `middle_school_questions.json` 文件

### 方法2：手动使用题目模板

如果无法运行脚本，可以：
1. 打开 `generate_middle_school_questions.js`
2. 找到 `questionTemplates` 对象
3. 手动提取每个知识点的题目模板
4. 按照JSON格式组织数据

## 题目字段说明

### 选择题格式

```json
{
  "_id": "唯一ID",
  "text": "题目文本",
  "answer": "A/B/C/D",
  "grammarPoint": "三级知识点名称",
  "category": "二级知识点名称",
  "type": "choice",
  "options": ["A. 选项1", "B. 选项2", "C. 选项3", "D. 选项4"],
  "analysis": "详细解析",
  "difficulty": "easy/medium/hard",
  "province": "云南",
  "year": 2025,
  "source": "题库",
  "schoolLevel": "middle",
  "examFrequency": "⭐⭐⭐",
  "examYears": [2023, 2024, 2025]
}
```

### 填空题格式

```json
{
  "_id": "唯一ID",
  "text": "题目文本（包含______）",
  "answer": "正确答案",
  "grammarPoint": "三级知识点名称",
  "category": "二级知识点名称",
  "type": "fill_blank",
  "analysis": "详细解析",
  "difficulty": "easy/medium/hard",
  "province": "云南",
  "year": 2025,
  "source": "题库",
  "schoolLevel": "middle",
  "examFrequency": "⭐⭐⭐",
  "examYears": [2023, 2024, 2025]
}
```

## 重要字段说明

- **source**: 使用 `"题库"` 区分于 `"变式题"`
- **examFrequency**: 基于2023-2025年真题统计的考频（⭐到⭐⭐⭐⭐⭐）
- **examYears**: 该知识点在真题中出现的年份数组
- **options**: 选择题必须包含此字段，格式为字符串数组

## 考频统计规则

- ⭐⭐⭐⭐⭐: 5次及以上
- ⭐⭐⭐⭐: 3-4次
- ⭐⭐⭐: 2次
- ⭐⭐: 1次
- ⭐: 未出现

## 题目统计

- **总题目数**: 105道
- **选择题**: 70道（35个知识点 × 2）
- **填空题**: 35道（35个知识点 × 1）

## 注意事项

1. 所有题目的 `source` 字段都是 `"题库"`，以区分于变式题
2. 考频信息基于提供的2023-2025年真题数据
3. 题目难度统一设置为 `"medium"`，可根据需要调整
4. 所有题目都包含完整的解析（analysis字段）
