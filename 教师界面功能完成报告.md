## 技术架构

### 前端技术栈
- **微信小程序框架**：原生小程序开发
- **页面结构**：WXML + WXSS + JavaScript
- **组件化设计**：可复用组件开发
- **状态管理**：页面级状态管理

### 数据管理
- **本地存储**：wx.getStorageSync / wx.setStorageSync
- **云端存储**：微信云开发数据库
- **数据同步**：本地+云端双重备份
- **数据恢复**：云端数据恢复机制

### 核心功能模块
1. **作业管理系统**：创建、发布、管理作业
2. **材料管理系统**：PPT/学案生成和管理
3. **点名器系统**：随机点名、统计功能
4. **摸底测试系统**：全语法点测试
5. **云端备份系统**：数据备份和恢复
