# 📄 习题导出功能使用说明

## 🎯 功能概述

习题导出功能专为教师备课设计，支持将练习题目导出为PDF、Word、Excel格式，方便教师制作教学材料。

## ✨ 主要特性

### 1. 多种导出格式
- **PDF格式**：适合打印，排版美观
- **Word格式**：便于编辑修改
- **Excel格式**：数据表格，便于统计分析

### 2. 多种导出模板
- **标准模板**：题目+答案+解析，适合教师参考
- **练习模板**：仅题目，答案另页，适合学生练习
- **测试模板**：题目+答题卡，适合考试使用

### 3. 智能筛选功能
- **难度筛选**：可选择简单、中等、困难题目
- **数量控制**：自定义导出题目数量
- **实时预览**：显示预计页数和文件大小

### 4. 用户权限管理
- **免费用户**：每日3次导出，每次最多20题
- **VIP用户**：每日10次导出，每次最多50题
- **教师认证**：每日20次导出，每次最多100题

## 🚀 使用方法

### 1. 打开导出功能
- 在练习页面右下角点击"📄 导出习题"按钮
- 系统会自动检查您的导出权限

### 2. 选择导出选项
- **导出格式**：选择PDF、Word或Excel
- **导出模板**：选择标准、练习或测试模板
- **题目数量**：调整要导出的题目数量
- **难度筛选**：选择题目难度级别

### 3. 预览和确认
- 查看预计页数和文件大小
- 确认导出内容包含的信息
- 检查剩余导出次数

### 4. 开始导出
- 点击"开始导出"按钮
- 等待文件生成完成
- 自动下载到本地

## 📋 导出模板说明

### 标准模板
```
题目内容
A. 选项A
B. 选项B
C. 选项C
D. 选项D
答案：C
解析：详细解析内容...
```

### 练习模板
```
题目内容
A. 选项A
B. 选项B
C. 选项C
D. 选项D

（答案另页显示）
```

### 测试模板
```
题目内容
A. 选项A    B. 选项B
C. 选项C    D. 选项D

答题卡：
题号：___ 答案：___
```

## 💡 使用建议

### 1. 备课使用
- 使用**标准模板**导出，包含答案和解析
- 选择**PDF格式**，便于打印和分享
- 建议每次导出20-50题，便于课堂使用

### 2. 学生练习
- 使用**练习模板**导出，答案另页
- 选择**Word格式**，便于编辑和调整
- 可以按难度筛选，循序渐进

### 3. 考试使用
- 使用**测试模板**导出，包含答题卡
- 选择**PDF格式**，保证格式统一
- 建议导出50-100题，满足考试需求
