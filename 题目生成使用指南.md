# 初中模块题目生成使用指南

## 快速开始

脚本 `generate_middle_school_questions.js` 已经包含了所有35个三级知识点的完整题目模板。

### 运行步骤

1. **确保Node.js环境**
   ```bash
   node --version
   ```

2. **运行生成脚本**
   ```bash
   node generate_middle_school_questions.js
   ```

3. **检查生成的文件**
   - 文件：`middle_school_questions.json`
   - 包含105道题目（70道选择题 + 35道填空题）

## 如果脚本无法运行

如果遇到路径或模块加载问题，可以：

1. **检查语法数据文件路径**
   - 确保 `miniprogram/config/middle-school-grammar-data.js` 存在
   - 脚本中使用了 `require('./miniprogram/config/middle-school-grammar-data.js')`

2. **修改require路径**
   - 根据实际项目结构调整路径

3. **手动提取题目**
   - 打开 `generate_middle_school_questions.js`
   - 找到 `questionTemplates` 对象（约第60-1000行）
   - 手动复制题目内容到JSON文件

## 题目格式验证

生成后，请检查：

- ✅ 每个知识点有2个选择题和1个填空题
- ✅ 选择题包含 `options` 字段（数组格式）
- ✅ 填空题的 `text` 字段包含 `______` 或 `(单词)`
- ✅ `source` 字段为 `"题库"`（不是 `"变式题"`）
- ✅ `examFrequency` 字段存在且格式正确
- ✅ `examYears` 字段为数组格式
