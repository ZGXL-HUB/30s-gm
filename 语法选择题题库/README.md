# 📚 语法选择题题库

## 📖 项目简介

这是一个为微信小程序项目生成的语法选择题题库，包含了60个语法小点中的58道精选题目。每道题目都是从原始填空题转换而来，保持了原有的题干和正确答案，并添加了合理的干扰项。

## 🎯 题库特点

- **全面覆盖**: 涵盖介词、代词、连词、冠词、名词、动词、谓语、非谓语、形容词、副词、定语从句、状语从句等12个语法大类
- **随机抽取**: 每个语法小点随机抽取一题，确保题目的代表性
- **智能干扰**: 根据答案类型生成合理的干扰项，提高题目的质量
- **选项随机**: 正确答案的位置随机分布，避免固定模式
- **详细解析**: 保留原有的详细解析，帮助学习者理解语法规则

## 📁 文件说明

### 主要文件

1. **`语法选择题题库.json`** - 完整的题库数据，包含所有题目、选项、答案和解析
2. **`语法选择题题库.html`** - 美观的HTML版本，适合在线查看和打印
3. **`语法选择题题库.md`** - Markdown版本，适合文档编辑和分享
4. **`生成语法选择题.js`** - 生成脚本，可以重新运行生成新的题库

### 数据结构

每道题目的数据结构如下：

```json
{
  "id": "唯一标识符",
  "grammarPoint": "语法点名称",
  "question": "题目内容",
  "options": [
    {
      "label": "A",
      "text": "选项内容",
      "isCorrect": false
    }
  ],
  "correctAnswer": "正确答案",
  "correctOption": "正确选项标签",
  "analysis": "详细解析",
  "category": "分类信息"
}
```

## 🚀 使用方法

### 1. 直接使用

- **HTML版本**: 用浏览器打开`语法选择题题库.html`，可以查看所有题目
- **JSON版本**: 可以导入到其他系统或应用中
- **Markdown版本**: 适合编辑和分享

### 2. 重新生成

如果需要重新生成题库（比如想要不同的随机题目），可以运行：

```bash
node 生成语法选择题.js
```

### 3. 集成到其他系统

JSON文件可以轻松集成到：

- 移动应用

- 教学管理系统
