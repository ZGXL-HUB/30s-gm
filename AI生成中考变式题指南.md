# AI生成中考真题变式题完整指南

## 一、推荐AI工具

### 🥇 首选推荐：Claude 3.5 Sonnet / Opus
**优势：**
- ✅ 对教育内容理解深入，语法准确性高
- ✅ JSON格式输出稳定，结构化能力强
- ✅ 支持长文本上下文，可一次性处理多题
- ✅ 变式题生成质量高，保持原题难度和考点

**使用平台：**
- Anthropic官网（Claude.ai）
- Cursor IDE（内置Claude）
- 其他支持Claude的API平台

### 🥈 次选推荐：GPT-4 / GPT-4 Turbo
**优势：**
- ✅ 通用性强，API调用方便
- ✅ 支持批量处理
- ✅ 输出格式可控

**劣势：**
- ⚠️ 语法细节可能不如Claude精准
- ⚠️ 需要更详细的提示词

**使用平台：**
- OpenAI官网
- 各种API代理平台
- Cursor IDE（内置GPT-4）

### 🥉 备选方案：DeepSeek / 文心一言
**优势：**
- ✅ 成本较低
- ✅ 中文理解好

**劣势：**
- ⚠️ JSON格式输出需要多次调整
- ⚠️ 语法准确性需要人工校验

## 二、数据结构要求

### 2.1 云数据库题目格式（必须匹配）

```json
{
  "_id": "自动生成",
  "text": "题目文本（包含题干和选项，如果是选择题）",
  "answer": "答案（选择题为A/B/C/D，填空题为具体答案）",
  "grammarPoint": "语法点名称（三级知识点，如'人称代词'）",
  "category": "语法分类（二级知识点，如'代词'）",
  "type": "题目类型（'choice'或'fill_blank'）",
  "analysis": "详细解析",
  "difficulty": "难度（'easy'/'medium'/'hard'）",
  "province": "省份（如'云南'）",
  "year": "年份（如2023/2024/2025）",
  "source": "来源（如'中考真题'/'变式题'）",
  "createdAt": "创建时间（ISO格式）"
}
```

### 2.2 初中模块知识点体系

**一级模块（6个）：**
- 基础词法
- 动词系统
- 复合句
- 句子结构
- 特殊句式
- 其他

**二级知识点（25个）：**
- 名词、代词、冠词、数词、介词、连词、形容词/副词
- 时态、语态、情态动词、非谓语动词
- 宾语从句、定语从句、状语从句
- 主谓一致、倒装句、强调句
- 等等

**三级知识点（具体语法点）：**
- 如：人称代词、物主代词、反身代词、不定代词
- 如：一般现在时、一般过去时、现在完成时
- 等等

## 三、准备工作清单

### ✅ 必须提供的材料

1. **原题完整信息**
   - 题目文本（含选项）
   - 正确答案
   - 原题解析
   - 原题年份（2023/2024/2025）
   - 原题省份（云南）

2. **知识点映射表**
   - 原题对应的二级知识点（如"代词"）
   - 原题对应的三级知识点（如"人称代词"）
   - 考频信息（⭐⭐⭐）
   - 年份标签（该知识点在哪些年份考过）

3. **题型信息**
   - 选择题 or 填空题
   - 如果是选择题，需要选项数量（通常是3个选项）

4. **难度参考**
   - 原题难度（easy/medium/hard）
   - 中考整体难度水平

5. **变式题要求**
   - 需要生成的数量（如20题）
   - 变式程度（保持原考点但改变语境/词汇/句式）

### 📋 额外准备工作

1. **建立知识点映射表**
   ```json
   {
     "原题知识点": {
       "category": "二级知识点",
       "grammarPoint": "三级知识点",
       "examFrequency": "⭐⭐⭐",
       "years": [2023, 2024, 2025]
     }
   }
   ```

2. **准备题目模板**
   - 选择题格式模板
   - 填空题格式模板
   - 解析格式模板

3. **建立质量检查清单**
   - 语法正确性
   - 难度一致性
   - 格式规范性
   - 知识点匹配度

### 4.1 基础版提示词（单题生成）

```

1. **保持原题核心考点**：变式题必须考查相同的语法知识点
2. **改变题目语境**：使用不同的词汇、句式、场景，但保持语法点不变
3. **难度一致**：变式题难度应与原题相当
4. **格式规范**：严格按照JSON格式输出

## 原题信息

## 五、注意事项与常见问题

### ⚠️ 必须注意的问题

1. **知识点映射准确性**
   - ❌ 错误：原题考查"人称代词"，但映射为"代词"（二级知识点）
   - ✅ 正确：category="代词", grammarPoint="人称代词"

2. **题型区分**
   - 选择题：type="choice"，answer为选项字母
   - 填空题：type="fill_blank"，answer为具体答案

3. **JSON格式规范**
   - 所有字符串必须用双引号
   - 数字不要加引号
   - 数组和对象格式正确
   - 不要有尾随逗号

4. **难度一致性**
   - 中考整体难度：medium为主
   - 简单题：easy（基础语法点）
   - 难题：hard（复合语法点）

5. **解析质量**
   - 必须解释为什么选这个答案
   - 说明考查的语法点
   - 选择题要解释干扰项错误原因

### 🔧 常见问题处理

**问题1：AI生成的JSON格式不正确**
- 解决：要求AI"只输出JSON，不要添加任何解释"
- 使用JSON验证工具检查
- 分批次生成，每次少量题目

**问题2：变式题难度不一致**
- 解决：在提示词中明确难度要求
- 提供难度参考标准
- 人工审核并调整

**问题3：知识点映射错误**
- 解决：提供详细的知识点映射表
- 在提示词中明确要求
- 生成后人工校验

**问题4：题目质量不高**
- 解决：使用更高质量的AI模型（Claude 3.5）
- 提供更多示例
- 分步骤生成：先生成题目，再生成解析

**问题5：批量生成效率低**
- 解决：使用批量生成提示词
- 分批处理，每批10-20题
- 使用API调用，自动化处理
