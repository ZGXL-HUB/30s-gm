## 使用说明

### 1. 访问教师界面
```javascript
// 跳转到教师工作台
wx.navigateTo({
  url: '/pages/teacher-dashboard/index'
});
```

### 2. 使用数据管理工具
```javascript
const teacherDataManager = require('../../utils/teacherDataManager');

// 创建班级
const newClass = teacherDataManager.classManager.createClass({
  name: '高三(1)班',
  description: '重点班',
  grade: '高三',
  maxStudents: 30
});

// 获取班级列表
const classes = teacherDataManager.classManager.getAllClasses();
```

### 3. 使用UI组件
```xml
<!-- 在wxml中使用卡片组件 -->
<teacher-card
  title="布置作业"
  subtitle="智能推荐、高考配比、专题练习"
  icon="📝"
  color="#667eea"
  bindtap="onCardTap"
/>
```

## 数据流程

### 1. 作业创建流程
1. 选择作业类型（智能推荐/高考配比/专题/自选）
2. 配置作业参数（乱序、数量、难度）
3. 预览作业内容
4. 发布作业到班级

### 2. 材料生成流程
1. 基于作业结果分析
2. 自动生成配套PPT/学案
3. 支持模板应用
4. 材料下载和管理

### 3. 班级管理流程
1. 创建班级和模板
2. 添加学生信息
3. 分配作业和材料
4. 数据统计和分析
