# 名词笔记示例

## 一、名词的分类

名词可以根据不同的标准进行分类，以下是详细的分类表格：

| 举例(中文 + 英文) | 类别名称 |
|---------------------|----------|
| 李白(Li Bai)、北京(Beijing)、春节(Spring Festival) | 专有名词 |
| 学生(student)、电脑(computer)、树(tree) | 个体名词(可数) |
| 家庭(family)、团队(team)、班级(class) | 集体名词(可单可复) |
| 水(water)、钢铁(steel)、空气(air) | 物质名词(不可数) |
| 幸福(happiness)、勇气(courage)、知识(knowledge) | 抽象名词(不可数) |

## 二、名词的识别与书写(后缀示例)

名词后缀是识别和构成名词的重要方法：

| 举例(动词 / 形容词→名词) | 名词后缀 |
|---------------------------|----------|
| 失败(fail→failure)、压力(press→pressure) | -ure |
| 死亡(die→death)、真相(true→truth) | -th |
| 发展(develop→development)、管理(manage→management) | -ment |
| 行动(act→action)、讨论(discuss→discussion) | -ion |
| 善良(kind→kindness)、黑暗(dark→darkness) | -ness |
| 拒绝(refuse→refusal)、批准(approve→approval) | -al |

## 三、时态标志词识别

时态标志词是识别英语时态的重要依据：

| 时态 | 时间标志词 (举例) | 核心功能 |
|------|------------------|----------|
| 一般现在时 | every day, usually, on Sundays | 经常性动作/客观事实 |
| 一般过去时 | yesterday, last week, ago | 过去具体时间点的动作 |
| 一般将来时 | tomorrow, next month, in the future | 将来时间的动作 |
| 现在进行时 | now, at the moment, look | 现在正在进行的动作 |
| 过去进行时 | at 8 pm yesterday, when I called | 过去某刻正在进行的动作 |
| 现在完成时 | for 3 years, since 2020, already | 过去动作对现在的影响/持续 |
| 过去完成时 | by last year, before he left | "过去的过去"的动作 |
| 将来进行时 | at 5 pm tomorrow, this time next week | 将来某刻正在进行的动作 |

## 四、常见考查形式

### 1. 作为词块考查(与介词、冠词、数词、形容词搭配)

考查重点：名词的书写(拼写正确性)、复数形式变化。

**(1)复数信号词**
- 数词提示：前面有 one 以外的数词(如 two, three, five 等)。
  例：three books(三本书)、five cities(五个城市)
- 修饰词提示：被 many, different, several, a lot of 等词修饰。
  例：many heroes(许多英雄)、different knives(不同的刀)

**(2)复数变化规则总结：**
- 一般情况加 -s(book→books)；
- 以 -s, -x, -ch, -sh 结尾加 -es(bus→buses)；
- 辅音字母 + y 结尾，变 y 为 i 加 -es(city→cities)；
- 元音字母 + y 结尾直接加 -s(boy→boys)；
- 以 -f/-fe 结尾，多数变 f/fe 为 v 加 -es(leaf→leaves)；
- 以 -o 结尾：有生命加 -es(potato→potatoes)，无生命加 -s(photo→photos)；
- 不规则变化(man→men, child→children)。

**考查示例：**
用所给词的适当形式填空：
There are three ______(knife)on the table.
解析：knife 以 -fe 结尾，变 fe 为 v 加 -es，答案为 knives。

### 2. 与连词考查(词性和形式一致)

考查重点：并列结构中名词的形式一致(单复数、词性匹配)。

**考查示例：**
填空：She bought some apples, ______(banana)and oranges.
解析：连词 and 连接并列名词，形式需一致(均为复数)，答案为 bananas。

### 3. 与动词考查(主谓一致)

考查重点：名词单复数决定动词形式(单数名词配单数动词，复数名词配复数动词)。

**考查示例：**
选择：The family ______(be)watching TV now.
解析：family 此处指 "家人"(复数概念)，动词用复数，答案为 are。

### 4. 名词所有格考查

考查重点：'s 所有格和 of 所有格的用法。

**考查示例：**
填空：This is ______(Tom)book.
解析：表示 "汤姆的书"，有生命的名词用 's 所有格，答案为 Tom's。
