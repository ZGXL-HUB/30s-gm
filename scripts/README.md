# Scripts 目录说明

本目录用于存放项目相关的脚本文件，按功能分类组织。

## 目录结构

- `legacy/` - 已废弃或不再使用的脚本
- `deploy/` - 部署相关脚本
- `data-analysis/` - 数据分析脚本
- `test/` - 测试脚本

## 使用说明

请将根目录下的临时脚本文件移动到对应的子目录中：
- 测试脚本 (`test_*.js`) → `test/`
- 修复脚本 (`fix_*.js`) → `legacy/` (如果不再使用)
- 分析脚本 (`analyze_*.js`) → `data-analysis/`
- 部署脚本 (`deploy_*.sh`, `deploy_*.ps1`) → `deploy/`

## 注意事项

移动脚本前请确认：
1. 脚本是否还在使用
2. 脚本的依赖关系
3. 是否需要更新脚本中的路径引用
