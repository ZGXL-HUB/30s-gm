# 高中模块与初中模块用户使用流程与功能分析

## 一、高中模块用户使用流程

### 1.1 入口流程
**步骤1：主界面选择**
- 用户进入小程序主界面
- 看到"高中"和"初中"两个入口卡片
- 点击"高中"卡片进入高中模块

**步骤2：作业类型选择**
- 进入高中作业布置页面
- 显示三种作业模式选择器：
  - 📚 **高考配比10题**：按高考比例配置题目
  - 🎯 **专题20题**：选择语法大点专题练习
  - ✏️ **自选20题**：自由选择语法小点

---

### 1.2 模式一：高考配比10题

#### 流程步骤：
1. **选择模式** → 点击"高考配比10题"
2. **系统自动组合** → 自动执行`executeGaokaoSystemCombo()`
   - 系统自动选择12个语法分类中的10个
   - **必选**：谓语、非谓语（2题）
   - **随机选择**：从其余10个分类中选择8个
   - 每个分类1题，共10题
3. **显示结果** → 自动显示选中的10个语法点标签
4. **变式题选择**（可选）→ 弹出变式题数量选择器（0-3题）
5. **预览生成** → 点击"我选好了"进入预览页面
6. **预览与复制** → 查看学生版/教师版，复制内容

#### 功能特点：
- ✅ **自动化程度高**：一键生成，无需手动选择
- ✅ **符合高考比例**：基于高考真题配比规则
- ✅ **支持变式题**：可选择0-3道变式题
- ✅ **快速高效**：适合快速组卷需求

#### 优势：
- 🟢 **操作简单**：只需点击一次，系统自动完成
- 🟢 **专业性强**：基于真实高考配比数据
- 🟢 **时间成本低**：30秒内完成组卷
- 🟢 **标准化**：每次生成都符合高考标准

#### 劣势：
- 🔴 **灵活性低**：无法自定义选择语法点
- 🔴 **随机性强**：每次生成的语法点组合不同
- 🔴 **无法微调**：不能调整单个语法点的题目数量
- 🔴 **依赖系统规则**：完全依赖预设的配比规则

---

### 1.3 模式二：专题20题

#### 流程步骤：
1. **选择模式** → 点击"专题20题"
2. **选择专题** → 从手风琴列表中选择1个语法大点（如"介词"、"代词"等）
   - 点击专题名称或复选框
   - 单选模式：选择新专题时自动取消之前的选择
3. **自动分配** → 系统自动将20题均分到该专题下的所有子知识点
4. **直接跳转** → 选择专题后自动跳转到预览页面
5. **预览与复制** → 查看学生版/教师版，复制内容

#### 功能特点：
- ✅ **专题聚焦**：针对特定语法大点深入练习
- ✅ **自动均分**：20题自动分配到所有子知识点
- ✅ **快速跳转**：选择后立即进入预览
- ✅ **结构清晰**：手风琴式知识点展示

#### 优势：
- 🟢 **针对性强**：适合专题复习和强化训练
- 🟢 **操作简单**：只需选择一个专题
- 🟢 **覆盖全面**：自动覆盖专题下所有子知识点
- 🟢 **快速生成**：选择即生成，无需额外配置

#### 劣势：
- 🔴 **无法微调**：不能调整子知识点的题目分配
- 🔴 **题型单一**：不支持选择题/填空题分别控制
- 🔴 **固定数量**：固定20题，无法自定义总数
- 🔴 **无法组合**：不能同时选择多个专题

---

### 1.4 模式三：自选20题

#### 流程步骤：
1. **选择模式** → 点击"自选20题"
2. **展开知识点** → 点击专题旁的"+"展开子知识点列表
3. **选择知识点** → 点击具体的小知识点进行选择
   - 可多选不同专题下的知识点
   - 已选知识点显示在上方标签区域
4. **全选功能** → 可点击专题旁的"全选"按钮快速选择该专题所有知识点
5. **自动均分** → 系统自动将20题均分到所有已选知识点
6. **微调数量** → 可通过滑块或数字输入调整每个知识点的题目数量
   - 总题数固定为20题
   - 调整一个知识点数量时，其他知识点自动调整
7. **确认选择** → 点击"我选好了"进入预览页面
8. **预览与复制** → 查看学生版/教师版，复制内容

#### 功能特点：
- ✅ **灵活选择**：可自由选择任意知识点组合
- ✅ **智能均分**：自动将20题均分到已选知识点
- ✅ **支持微调**：可调整每个知识点的题目数量
- ✅ **实时反馈**：显示总题数和各知识点分配情况

#### 优势：
- 🟢 **灵活性高**：完全自定义知识点组合
- 🟢 **精准控制**：可微调每个知识点的题目数量
- 🟢 **适合个性化**：满足不同教学需求
- 🟢 **操作直观**：标签显示清晰，调整方便

#### 劣势：
- 🔴 **操作复杂**：需要手动选择多个知识点
- 🔴 **时间成本高**：相比其他模式耗时更长
- 🔴 **题型单一**：不支持选择题/填空题分别控制
- 🔴 **固定总数**：总题数固定为20题，无法自定义

---

## 二、初中模块用户使用流程

### 2.1 入口流程
**步骤1：主界面选择**
- 用户进入小程序主界面
- 看到"高中"和"初中"两个入口卡片
- 点击"初中"卡片进入初中模块

**步骤2：作业类型选择**
- 进入初中作业布置页面（绿色导航栏）
- 显示三种作业模式选择器：
  - 📚 **中考配比15题**：10选择+5填空，按中考比例配置
  - 🎯 **专题20题**：选择语法大点专题练习
  - ✏️ **自选20题**：自由选择语法小点

---

### 2.2 模式一：中考配比15题

#### 流程步骤：
1. **选择模式** → 点击"中考配比15题"
2. **年份套餐选择** → 选择年份套餐（2023/2024/2025）
   - 显示该年份考过的知识点
   - 未考过的知识点显示为灰色（不可选）
3. **系统自动组合** → 自动执行`executeZhongkaoSystemCombo()`
   - 根据中考配比规则自动选择知识点
   - 10道选择题 + 5道填空题
   - 按中考真题比例分配
4. **显示结果** → 显示选中的知识点和题型分配
5. **预览生成** → 点击"进入预览"进入预览页面
6. **预览与复制** → 查看学生版/教师版，复制内容

#### 功能特点：
- ✅ **年份套餐**：可选择不同年份的真题配比
- ✅ **双题型控制**：选择题和填空题分别控制
- ✅ **考情标注**：显示知识点考频和年份信息
- ✅ **符合中考**：基于云南中考真题配比

#### 优势：
- 🟢 **贴近真题**：基于真实中考真题数据
- 🟢 **年份可选**：支持不同年份的配比方案
- 🟢 **题型区分**：选择题和填空题分别控制
- 🟢 **考情透明**：显示考频和年份信息

#### 劣势：
- 🔴 **实现不完整**：当前为占位实现，需要完善
- 🔴 **灵活性低**：无法自定义选择知识点
- 🔴 **依赖数据**：需要完整的真题数据支撑
- 🔴 **无法微调**：不能调整单个知识点的题目数量

---

### 2.3 模式二：专题20题（初中版）

#### 流程步骤：
1. **选择模式** → 点击"专题20题"
2. **题型数量设置** → 设置选择题和填空题数量
   - 选择题：5/10/15（按钮选择）
   - 填空题：5/10/15（按钮选择）
   - 实时显示总题数和题型分布
3. **选择专题** → 从手风琴列表中选择1个语法大点
   - 显示考频星星（⭐⭐⭐）
   - 显示年份标签（如"2023, 2024"）
   - 显示占比信息（如"26.7%"）
4. **智能均分** → 系统自动将题目均分到该专题下的所有子知识点
   - 选择题和填空题分别均分
   - 显示每个知识点的题型分配
5. **微调分配** → 可调整每个知识点的题型数量
   - 选择题：+/- 按钮调整
   - 填空题：+/- 按钮调整
   - 实时更新总题数
6. **确认生成** → 点击"进入预览"进入预览页面
7. **预览与复制** → 查看学生版/教师版，复制内容

#### 功能特点：
- ✅ **双题型控制**：选择题和填空题独立设置
- ✅ **智能均分**：自动均分到所有子知识点
- ✅ **支持微调**：可调整每个知识点的题型分配
- ✅ **考情展示**：显示考频、年份、占比等信息

#### 优势：
- 🟢 **题型灵活**：选择题和填空题可分别控制
- 🟢 **分配智能**：自动均分，支持微调
- 🟢 **信息丰富**：考情数据一目了然
- 🟢 **操作直观**：按钮式选择，调整方便

#### 劣势：
- 🔴 **操作步骤多**：需要先设置题型，再选择专题
- 🔴 **固定总数**：总题数由题型设置决定，无法直接设置总数
- 🔴 **无法组合**：不能同时选择多个专题
- 🔴 **学习成本**：需要理解题型分配逻辑

---

## 四、整体优劣势总结

### 4.1 高中模块整体优势

1. **操作简单**：三种模式都相对简单，学习成本低
2. **快速高效**：特别是高考配比模式，一键生成
3. **变式题支持**：高考配比模式支持变式题功能
4. **成熟稳定**：功能经过测试，稳定性好

### 4.2 高中模块整体劣势

1. **功能单一**：不支持双题型控制
2. **灵活性低**：无法自定义题型分配
3. **信息不足**：缺少考情数据展示
4. **微调有限**：专题模式无法微调

### 4.3 初中模块整体优势

1. **功能丰富**：双题型控制、考情展示、微调功能
2. **贴近实际**：基于真题数据，更贴近中考
3. **信息透明**：考频、年份、占比等信息完整
4. **灵活性强**：支持题型和知识点的灵活组合

### 4.4 初中模块整体劣势

1. **操作复杂**：步骤较多，学习成本高
2. **实现不完整**：中考配比模式需要完善
3. **缺少变式题**：暂不支持变式题功能
4. **时间成本高**：完成一次组卷需要更多时间

## 五、改进建议

### 5.1 高中模块改进建议

1. **增加双题型控制**：在专题和自选模式中增加选择题/填空题分别控制
2. **增加考情信息**：显示知识点的重要性和考频
3. **优化微调功能**：专题模式增加微调功能
4. **增加年份选择**：高考配比模式可增加不同年份的配比方案

### 5.2 初中模块改进建议

1. **简化操作流程**：优化题型设置和知识点选择的交互
2. **完善中考配比**：完成中考配比模式的实现
3. **增加变式题支持**：在专题和自选模式中增加变式题功能
4. **优化信息展示**：考情信息展示更加直观
