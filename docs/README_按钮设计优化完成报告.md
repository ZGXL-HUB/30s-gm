## 设计规范

### 颜色主题
- **主色调**: #4096ff (蓝色)
- **渐变**: linear-gradient(135deg, #4096ff 0%, #1677ff 100%)
- **次要色**: #6c757d (灰色)
- **禁用色**: #d9d9d9 (浅灰色)

### 按钮规格
- **圆角**: 12rpx (统一标准)
- **内边距**: 16rpx 32rpx (标准尺寸)
- **字体大小**: 28rpx (标准)
- **字体粗细**: 500 (中等)

### 交互效果
- **过渡时间**: 0.3s ease
- **按压效果**: translateY(2rpx)
- **阴影**: 0 2rpx 8rpx rgba(0,0,0,0.1)
- **激活阴影**: 0 1rpx 4rpx rgba(64, 150, 255, 0.4)

## 最新修复 (2024年更新)

### 按钮显示问题修复
- **问题**: 右侧按钮被遮住右半边部分
- **原因**: 底部操作栏内边距不足，按钮宽度设置过大
- **解决方案**:
  1. 调整底部操作栏内边距从15rpx增加到32rpx
  2. 优化按钮内边距从32rpx减少到24rpx
  3. 设置按钮最小宽度100rpx，最大宽度140rpx
  4. 添加flex-shrink: 0防止按钮被压缩
  5. 优化actions容器和total-box的flex属性
  6. 添加响应式设计，适配不同屏幕尺寸

### 响应式设计优化
- **750rpx以下屏幕**: 调整按钮尺寸和字体大小
- **600rpx以下屏幕**: 进一步优化按钮尺寸和间距
- **确保在各种设备上按钮都能完整显示**
