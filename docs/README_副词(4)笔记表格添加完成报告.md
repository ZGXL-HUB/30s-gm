- 找副词修饰的词

### 练习题目
1. **Obviously** - 一般情况，形容词直接加 -ly 变副词
2. **Luckily** - 以"辅音字母 + y"结尾的形容词，变 y 为 i 再加 -ly
3. **Truly** - 以 -ue 结尾的形容词，去 e 加 -ly
4. **Scientifically** - 以 -ic 结尾的形容词，加 -ally
5. **Fully** - 以 l 结尾的形容词，加 -ly（双写 l）
