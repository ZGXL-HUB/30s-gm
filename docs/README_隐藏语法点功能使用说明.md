# 隐藏语法点功能使用说明

## 功能概述

隐藏语法点功能允许你控制哪些语法点不想让用户看到。被隐藏的语法点将不会在语法选择页面显示，用户也无法选择这些语法点进行练习。

## 实现方案

### 方案一：直接修改配置文件（推荐）

**文件位置：** `miniprogram/config/hidden-points.js`

**使用方法：**
1. 打开 `miniprogram/config/hidden-points.js` 文件
2. 在 `hiddenPoints` 数组中添加要隐藏的语法点名称
3. 可选：在 `hiddenPointsReason` 对象中添加隐藏原因

**示例：**
```javascript
const hiddenPoints = [
  "时态(过去将来时)",
  "语态(被动+八大时态)", 
  "插入语与动词",
  "f/fe结尾"
];

const hiddenPointsReason = {
  "时态(过去将来时)": "该语法点暂时下架维护",
  "语态(被动+八大时态)": "内容正在更新中",
  "插入语与动词": "题目数量不足，暂时隐藏"
};
```

### 方案二：使用管理页面

**页面路径：** `/pages/admin/hidden-points-manager/index`

**使用方法：**
1. 在微信开发者工具中访问该页面
2. 点击"添加"按钮添加要隐藏的语法点
3. 输入语法点名称和隐藏原因（可选）
4. 点击"保存配置"保存更改

## 注意事项

1. **语法点名称必须完全匹配**：隐藏语法点时，名称必须与上述列表中的名称完全一致
2. **立即生效**：修改配置文件后，重新编译小程序即可生效
3. **不影响已有数据**：隐藏语法点不会影响已有的题目数据，只是在前端不显示
4. **可随时恢复**：从隐藏列表中移除语法点名称即可恢复显示

## 技术实现

### 核心文件

1. **配置文件：** `miniprogram/config/hidden-points.js`
2. **语法选择页面：** `miniprogram/pages/grammar-select/index.js`
3. **自定义组合设置页面：** `miniprogram/pages/custom-combo-setting/index.js`
4. **管理页面：** `miniprogram/pages/admin/hidden-points-manager/`

### 实现原理

1. 在语法选择页面的WXML中使用条件渲染 `wx:if="{{!isHiddenPoint(point.name || point)}}"`
2. 在相关页面的JS中引入配置文件并检查语法点是否被隐藏
3. 被隐藏的语法点不会显示在用户界面上，也不会被包含在题目生成逻辑中

## 常见问题

**Q: 隐藏语法点后，用户之前选择的组合中如果包含这些语法点怎么办？**
A: 系统会自动过滤掉被隐藏的语法点，只使用可见的语法点生成题目。

**Q: 可以临时隐藏语法点吗？**
A: 可以，只需要在配置文件中添加或移除语法点名称即可。

**Q: 隐藏语法点会影响题目数据吗？**
A: 不会，隐藏功能只影响前端显示，不会修改或删除任何题目数据。

**Q: 如何批量隐藏多个语法点？**
A: 可以在配置文件的 `hiddenPoints` 数组中一次性添加多个语法点名称。
