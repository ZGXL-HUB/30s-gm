# 打卡海报与分享说明

## 一、打卡海报优化说明

小程序内已完成打卡海报的界面优化，包括：
- 渐变背景、顶部品牌色条
- 中央白卡 + 阴影
- 标题/副标题/完成时间/分隔线/底部文案
- 适合长按保存或分享的尺寸（375×500 逻辑像素）

若仍需**自行设计海报**（用于公众号、印刷或更精美分享图），可参考以下建议。

### 1. 推荐制作/搜索平台

| 平台 | 说明 |
|------|------|
| **Canva（可画）** | 模板多，有「学习/打卡/证书」类，支持中文 |
| **创客贴** | 国内模板，适合教育、打卡、日签 |
| **稿定设计** | 小程序/公众号尺寸模板多 |
| **Figma / 即时设计** | 自定义尺寸，适合做统一品牌规范 |

搜索关键词示例：`学习打卡`、`课程完成`、`日签`、`证书`、`教育海报`。

### 2. 推荐图片尺寸

- **竖版分享图（通用）**：750×1000 px（宽高比 3:4），或 750×1334 px（接近手机屏）
- **与小程序内一致**：375×500 px（导出时可用 2x：750×1000）
- **朋友圈九宫格**：单图建议 1080×1080 或 1080×1920

### 3. 建议包含的元素

- **主标题**：如「本课已全部完成」「每日打卡」
- **课程/课时信息**：如「第1课 · 翻译练习打卡」
- **完成时间**：如「2026年2月22日」
- **一句口号**：如「一起来学习吧～」「坚持每日打卡」
- **品牌/课程标识**：Logo 或课程名称
- **留白**：四周留白，避免文字贴边，方便在不同场景下使用

## 二、为什么无法分享到好友/朋友圈？需要怎么做？

### 1. 分享给「微信好友」

- **正确用法**：在完成页点击按钮「分享给好友 / 朋友圈」。
- **技术实现**：按钮使用 `open-type="share"`，会触发页面的 `onShareAppMessage`，弹出微信自带的分享面板，选择「发送给朋友」即可分享**小程序卡片**（标题 + 路径）。
- **若点击没反应，可能原因**：
  1. **在开发者工具中**：部分工具对 `open-type="share"` 支持不完整，请在**真机预览**或**正式版小程序**中测试。
  2. **页面未定义分享**：确认该页面（如翻译完成页）的 js 中有 `onShareAppMessage()` 并返回 `title`、`path`。
  3. **基础库版本**：确保微信基础库版本足够（一般 2.x 以上均支持）。

**您需要做的**：用手机微信扫码打开小程序，在「翻译完成」页点击「分享给好友 / 朋友圈」，应会弹出分享面板，选择好友即可。

### 2. 分享到「朋友圈」

- **正确用法**：用户需要点击小程序**右上角「…」**，在菜单里选择「**分享到朋友圈**」。
- **注意**：分享到朋友圈是否显示，由**微信与小程序后台配置**共同决定：
  1. **小程序后台**：登录 [微信公众平台](https://mp.weixin.qq.com) → 小程序 → 设置 → 基本设置 → 「分享到朋友圈」需已开启（若该选项存在）。
  2. **代码侧**：页面实现 `onShareTimeline()` 并返回 `title`、`query` 等，可自定义朋友圈分享的标题和查询参数。
  3. **部分账号/类目**：某些类目或账号类型可能不开放「分享到朋友圈」，需以实际菜单为准。

**您需要做的**：
1. 在真机中打开小程序，点击右上角「…」，查看是否有「分享到朋友圈」。
2. 若无，到公众平台检查是否开通该能力及类目是否支持。
3. 若有，可在我方页面中实现 `onShareTimeline`，以定制朋友圈标题和参数。

### 3. 小结

| 目标 | 操作 | 备注 |
|------|------|------|
| 分享给好友 | 点击完成页「分享给好友/朋友圈」→ 选「发送给朋友」 | 建议真机测试 |
| 分享到朋友圈 | 点击右上角「…」→「分享到朋友圈」 | 依赖后台与类目开通 |
| 分享海报图 | 先点「查看海报」→ 长按图片保存 → 在微信中发图 | 不依赖分享接口 |

若您希望分享时**默认带上海报图**，目前微信分享小程序卡片的主图主要由系统或后台配置决定；将海报生成后「查看海报」再长按保存或发送给好友，是当前最稳定的做法。
