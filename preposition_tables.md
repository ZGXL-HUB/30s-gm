## 表格4：介词(1)笔记表格 - 常见介词分类及用法

| 分类 | 介词 | 用法示例 |
|------|------|----------|
| 方位介词 | at | 用于小地点(具体位置点)，如 at the door(在门口)、at the corner(在角落) |
| 方位介词 | in | 用于大地点(空间内部)，如 in the room(在房间里)、in China(在中国) |
| 方位介词 | on | 用于表面接触，如 on the table(在桌上)、on the street(在街上) |
| 方位介词 | under | 在…… 下方，如 under the bed(在床底下) |
| 方位介词 | above | 在…… 上方(不接触)，如 above the clouds(在云层上方) |
| 方位介词 | below | 在…… 下方(不接触)，如 below the window(在窗户下方) |
| 方位介词 | beside | 在…… 旁边，如 beside the river(在河边) |
| 方位介词 | between | 在两者之间，如 between you and me(在你我之间) |
| 方位介词 | among | 在三者及以上之间，如 among the students(在学生们中) |
| 方位介词 | behind | 在…… 后面，如 behind the tree(在树后) |
| 方位介词 | in front of | 在…… 前面(外部)，如 in front of the house(在房子前) |
| 方位介词 | inside | 在…… 内部，如 inside the box(在盒子里) |
| 方位介词 | outside | 在…… 外部，如 outside the school(在学校外) |
| 时间介词 | at | 具体时间点，如 at noon(在正午)、at midnight(在午夜) |
| 时间介词 | in | 较长时间(年 / 月 / 季节 / 世纪)，如 in spring(在春天)、in the 21st century(在 21 世纪) |
| 时间介词 | on | 具体某天及某天的片段，如 on Christmas Day(在圣诞节)、on a rainy afternoon(在一个下雨的下午) |
| 时间介词 | after | 在…… 之后，如 after class(课后)、after 2020(2020 年后) |
| 时间介词 | before | 在…… 之前，如 before dinner(晚饭前)、before 8 o'clock(8 点前) |
| 时间介词 | by | 到…… 为止，如 by Friday(到周五为止) |
| 时间介词 | during | 在…… 期间，如 during the holiday(假期期间) |
| 时间介词 | for | 持续一段时间，如 for three years(持续三年) |
| 时间介词 | since | 自从…… 以来，如 since 2018(自从 2018 年以来) |
| 其他介词 | about | 关于，如 talk about the film(谈论那部电影) |
| 其他介词 | against | 反对；靠着，如 fight against injustice(反对不公)、lean against the wall(靠着墙) |
| 其他介词 | with | 伴随；用；和…… 一起，如 walk with a dog(带着狗散步)、cut with a knife(用刀切) |
| 其他介词 | without | 没有，如 go out without a coat(没穿外套出去) |
| 其他介词 | for | 为了；因为，如 study for the exam(为考试学习)、cry for joy(因高兴而哭) |
| 其他介词 | from | 来自；从，如 be from America(来自美国)、learn from mistakes(从错误中学习) |
| 其他介词 | to | 向；对，如 write to a friend(给朋友写信)、be kind to others(对他人友善) |
| 其他介词 | of | …… 的；关于，如 a map of China(中国地图)、think of a plan(想出一个计划) |
| 其他介词 | by | 通过；被，如 by bus(乘公交)、be loved by everyone(被所有人喜爱) |
| 其他介词 | because of | 因为(后接名词)，如 late because of the rain(因雨迟到) |
| 其他介词 | instead of | 代替，如 go by bike instead of by car(骑自行车代替开车) |

## 表格6：介词(3)笔记表格 - 常见名词后缀、介词后的动词ing书写

| 介词 | 含义(后接名词 / 代词时) | 考察示例 |
|------|------------------------|----------|
| at | 接具体时间点、小地点 | He arrived ______ the airport at 3 p.m.(答案：at) |
| in | 接年 / 月 / 季节、大地点、时间段 | We will visit Xi'an ______ July.(答案：in) |
| on | 接具体日期、星期、具体某天的早 / 中 / 晚 | They have a meeting ______ Monday morning.(答案：on) |
| with | 接工具、伴随的人或物 | She writes letters ______ a pencil.(答案：with) |
| for | 接对象、目的、时间段 | I waited ______ you for an hour.(答案：for) |

| 名词后缀 | 例词 | 介词 + 该名词考察示例 |
|----------|------|----------------------|
| -tion | education | She is interested ______ education.(答案：in) |
| -ment | development | We talked ______ the development of the city.(答案：about) |
| -ity | ability | He has a talent ______ ability in music.(答案：for) |
| -ness | happiness | They dreamed ______ happiness.(答案：of) |
| -sion | discussion | We had a discussion ______ the plan.(答案：on) |

| 规则 | 动词原形 | 现在分词形式 |
|------|----------|--------------|
| 一般情况直接加 - ing | work | working |
| 以不发音的 e 结尾，去 e 加 - ing | take | taking |
| 以重读闭音节结尾，双写尾字母加 - ing | run | running |
| 以 ie 结尾，改 ie 为 y 加 - ing | lie | lying |
| 以 y 结尾(非 ie)，直接加 - ing | play | playing |
